Advanced Inventory Management System

A.I.M.S. allows opticians to manage their inventory, track sales, see missing items, and even locate frames using RFID technology.

Features

  • Inventory
    • Sends a report of inventory status via Email
    • Audit items
    • Move items to locations
  • Assign Tag
    • Commission RFID tags with Price
  • Locate items
  • Sell items from inventory
  • Onboard Universal Product Code
