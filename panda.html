Panda Studio / Workshop Quantium

An Integrated Development Environment for programming Human-Machine Interface Displays.

Features

  • Create and customize widgets.
  • Design user interface for HMI displays of 4D Systems with a WYSIWYG interface.
  • Add logic to the UI with the code editor.
  • Ability to program the HMI displays to control its internal and external hardware peripherals.
