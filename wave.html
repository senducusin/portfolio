Wave: Enterprise Inventory

Wave allows users to perform audits and inventories by connecting and sharing data to a Pogi server.

Features

    • Move Items to a different location
    • Continous Inventory Stream
    • View Item
      • Update Item
  • Commission Item
    • Single Commission
    • Batch Commission
  • Search
    • Locate Item via RFID
    • Search Item in Database
